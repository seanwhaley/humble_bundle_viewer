# Installation

## Prerequisites

- Python 3.10+
- Internet access to install packages and browser binaries

## Install steps

1. Create and activate a virtual environment (recommended).

2. Install dependencies: `pip install -r requirements.txt`.

3. Install browser binaries for `Playwright`: `python -m playwright install`.

4. Install the package in editable mode: `pip install -e .`.
