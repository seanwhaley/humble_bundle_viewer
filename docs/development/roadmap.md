# Roadmap

Planned improvements for future releases.

Use this page for release-scale priorities and longer-horizon direction. Keep active execution details in `docs/development/work-efforts/<effort-name>/` while work is in progress.

## Near term

- add automated frontend tests for core viewer workflows
- resolve known category-route rendering issues and add regression coverage
- improve release automation and docs verification in CI
- tighten accessibility and responsive polish across data-heavy routes

## Later

- add resume support for interrupted managed sync workflows
- expand filtering and export workflows for large libraries
- evaluate packaging improvements for easier first-time local setup
